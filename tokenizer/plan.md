# why
- 원하는 대로 단어가 나오지 않는 KSL이나 그저 토큰화로는 구분할 수 없는 ISL을 위해 직접 토큰화 모델을 만듦
# how
1. 이미 있는 토크나이저중 사용자 사전 추가가 가능한것들(MeCab)에 사전의 단어를 전부 가져다가 넣어버리자.
2. 글자단위로 S2S를 사용해 해당 문서 내에 들어있는 토큰을 추출
# structure
1. 사용자 사전 추가가 가능한 토크나이저 탐색, 사전 추가/교체.
2. S2S제작/사용
3. 동음이의어 문제 
# add func
1. 실시간 녹음/번역
2. 자동으로 화면 전환
